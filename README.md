## Instructions
Create the following folders within CAworkshop folder:
- /data/raster (inside this, copy the PISCOp_v2.1.1.nc and SRTM_10km.nc files).
- /data/vector (inside this, copy the basins.gpkg file).
- /info
- /scripts
- /exports
